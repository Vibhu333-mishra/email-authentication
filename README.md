# Email Authentication Website

A modern and responsive email authentication web application built with **React** and **Vite**. The application provides a secure and user-friendly interface for user registration, login, and authentication.

## 🚀 Features

* User Registration (Sign Up)
* User Login (Sign In)
* Email Authentication
* Form Validation
* Responsive User Interface
* Fast Performance with Vite

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/email-authentication-app.git
```

2. Navigate to the project folder

```bash
cd email-authentication-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

The application will run at:

```
http://localhost:5173
```
